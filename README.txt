# Сайт-визитка (мини-презентация)

## Как открыть
1) Откройте файл `index.html` двойным кликом — он запустится в браузере.
2) Если хотите показывать как «презентацию» — нажмите кнопку **«Показать как презентацию»** (это откроет печать/preview, можно и в PDF сохранить).

## Как заменить контакты
В `index.html`:
- замените `hello@example.com` на ваш e-mail
- замените в блоке «Контакты» имя/телефон/telegram на ваши данные

## Как выложить в интернет (быстро)
- Netlify / Vercel / GitHub Pages: загрузите содержимое папки (index.html + assets) — и получите ссылку.

## Файлы-ассеты
- В репозитории нет бинарных ассетов (png и архив) — их нужно добавить самостоятельно в каталог `assets/`.
- При необходимости используйте исходные имена: `bg_marble_soft.png`, `cream.png`, `flower.png`, `mosaic.png`, `red.png`, `white.png`, а также `favicon.svg`.
